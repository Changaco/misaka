InstallationÂ¶

Misaka supports Python 2 and 3 (tested with 2.7 and 3.2). The only dependency is Cython, which is used to generate a large part of the C code. Misaka can be installed with easy_install or pip:

pip install misaka

The latest sources can be cloned from the Github repository and installed with:

python setup.py install
