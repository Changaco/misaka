API¶

This page describes all classes, functions and constants. Usage examples (renderer classes and pre- and post-processors) can be found at the Examples page.

Constants¶

Markdown parser extensions.

misaka.EXT_NO_INTRA_EMPHASIS¶

Do not parse emphasis inside of words. Strings such as foo_bar_baz will not generate <em> tags.

misaka.EXT_TABLES¶

Parse PHP-Markdown tables.

misaka.EXT_FENCED_CODE¶

Parse fenced code blocks, PHP-Markdown style. Blocks delimited with 3 or more ~ or backticks will be considered as code, without the need to be indented. An optional language name may be added at the end of the opening fence for the code block.

misaka.EXT_AUTOLINK¶

Parse links even when they are not enclosed in <> characters. Autolinks for the http, https and ftp protocols will be automatically detected. Email addresses are also handled, and http links without protocol, but starting with www.

misaka.EXT_STRIKETHROUGH¶

Parse strikethrough, PHP-Markdown style Two ~ characters mark the start of a strikethrough, e.g. this is ~~good~~ bad.

misaka.EXT_LAX_HTML_BLOCKS¶

HTML blocks do not require to be surrounded by an empty line as in the Markdown standard.

misaka.EXT_SPACE_HEADERS¶

A space is always required between the hash at the beginning of a header and its name, e.g. #this is my header would not be a valid header.

misaka.EXT_SUPERSCRIPT¶

Parse superscripts after the ^ character; contiguous superscripts are nested together, and complex values can be enclosed in parenthesis, e.g. this is the 2^(nd) time.

HTML render flags for HtmlRenderer and HtmlTocRenderer.

misaka.HTML_SKIP_HTML¶

Do not allow any user-inputted HTML in the output.

misaka.HTML_SKIP_STYLE¶

Do not generate any <style> tags.

misaka.HTML_SKIP_IMAGES¶

Do not generate any <img> tags.

misaka.HTML_SKIP_LINKS¶

Do not generate any <a> tags.

misaka.HTML_EXPAND_TABS¶

Unused.

misaka.HTML_SAFELINK¶

Only generate links for protocols which are considered safe.

misaka.HTML_TOC¶

Add HTML anchors to each header in the output HTML, to allow linking to each section.

misaka.HTML_HARD_WRAP¶

Insert HTML <br> tags inside on paragraphs where the origin Markdown document had newlines (by default, Markdown ignores these newlines).

misaka.HTML_USE_XHTML¶

Output XHTML-conformant tags.

misaka.HTML_ESCAPE¶

HTML_ESCAPE overrides SKIP_HTML, SKIP_STYLE, SKIP_LINKS and SKIP_IMAGES. It doens’t see if there are any valid tags, just escape all of them.

Constants for the html() function.

misaka.HTML_SMARTYPANTS¶

Post-process rendered markdown text with SmartyPants.

misaka.HTML_TOC_TREE¶

Render a table of contents.

Shorthand¶

misaka.html(text, extensions=0, render_flags=0)¶

Convert markdown text to (X)HTML:

misaka.html('source *text*',
    extensions=EXT_AUTOLINK|EXT_SUPERSCRIPT|EXT_STRIKETHROUGH,
    render_flags=HTML_SKIP_HTML|HTML_USE_XHTML)
Parameters:
  • text – text as a (unicode) string.
  • extensions – enable additional Markdown extensions with the EXT_* constants.
  • render_flags – adjust rendering behaviour with the HTML_* constants.

Post-processors¶

class misaka.SmartyPants¶

Smartypants post-processor for renderers. It can be used like this:

class BleepRenderer(HtmlRenderer, SmartyPants):
    pass
postprocess(text)¶

Process input text.

Renderers¶

class misaka.BaseRenderer(flags=0)¶

The BaseRenderer class does nothing by itself. It should be subclassed.

Parameters:flags – flags that can be used by the renderer.
setup()¶

The setup method can be overridden by a subclass. This method is executed when a new object of the class is created. Right after __init__.

class misaka.HtmlRenderer(flags=0)¶

A HTML renderer.

Parameters:flags – Accepts the HTML_* constants as flags.
class misaka.HtmlTocRenderer(flags=0)¶

A HTML table of contents renderer.

Parameters:flags – Accepts the HTML_* constants as flags.

The Markdown parser.

Parameters:

  • extensions – enable Markdown extensions with the EXT_* constants.
render(text)¶

Render the given source text.

Parameters:text – text as a (unicode) string.
