Misaka

index changelog manual api

A Python 2 and 3 binding for Sundown, a really fast Markdown parser implemented in C. Misaka is written in Cython and C. And it features a set of Markdown extensions and customizable renderers. Just like the Sundown binding for Ruby, Redcarpet.

Projects that use Misaka:

  • Jasy
  • mynt
  • Composer
  • Socrates
