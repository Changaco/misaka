Misaka¶

The Python binding for Sundown, a markdown parsing library.

Misaka is made with Cython and C and provides the same/similar functionality as Redcarpet, the Ruby binding for Sundown. The parser and renderer are decoupled. This means you can make your own renderers.

The documentation is also available on Read the Docs.
